# BlockPicker 色块

---

## 描述

这个组件仅用于展示当前颜色的效果。

---

## 示例:

```tsx
import React from 'react';
import { BlockPicker } from 'react-color-lite';

export default () => <BlockPicker color={{ rgb: { r: 26, g: 232, b: 104 } }} alpha={90} />;
```
