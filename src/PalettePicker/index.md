### PalettePicker 饱和度/明度

---

## 描述

该组件基于 HSV 模型，用于控制 饱和度（Saturation），明度（Value）两个色值

---

## 示例:

```tsx
import React from 'react';
import { PalettePicker } from 'react-color-lite';

export default () => <PalettePicker title="PalettePicker" />;
```
